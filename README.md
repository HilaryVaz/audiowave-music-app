# AudioWave Music Web App

AudioWave is a music streaming web application built using:

- PHP
- MySQL
- Bootstrap
- JavaScript

## Features

- Music library
- Search songs
- Play / Next / Previous player
- Favorites system
- Playlist management
- Lyrics display
- Upload songs

## Installation

1. Import the database file `music_streaming.sql` into MySQL.
2. Place the project folder inside `htdocs`.
3. Start Apache and MySQL.
4. Open:

http://localhost/MusicStreaming/music.php
